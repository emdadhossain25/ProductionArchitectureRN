# ML Quick Reference

## Three Learning Types

**Supervised**: Have answers, learn from labeled data
Examples: spam detection, price prediction

**Unsupervised**: No answers, find patterns
Examples: customer grouping, anomaly detection

**Reinforcement**: Learn from rewards/penalties
Examples: game playing, robotics

## Key Terms

**Dataset**: Your training examples
**Features**: Input variables (what you measure)
**Labels**: Output answers (what you predict)
**Model**: The learned patterns
**Training**: Learning process
**Prediction**: Using the model

## ML vs Traditional

Traditional: Write explicit rules
ML: Learn rules from data

Use ML when:
- Too many rules
- Rules change
- Patterns complex

## Workflow

1. Collect data
2. Prepare data
3. Choose algorithm
4. Train model
5. Evaluate accuracy
6. Deploy to production

## Mobile ML

On-device: Fast, private, offline
Cloud: Powerful, updates easily
