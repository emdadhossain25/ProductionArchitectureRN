# App Store Submission Checklist

## Pre-Submission Requirements

### Apple Developer Account
- [ ] Enrolled in Apple Developer Program ($99/year)
- [ ] Agreement accepted in App Store Connect
- [ ] Banking and tax info completed

### App Store Connect Setup
- [ ] App created in App Store Connect
- [ ] Bundle ID matches project
- [ ] App name available
- [ ] Primary language selected
- [ ] SKU created

### App Information
- [ ] App name (max 30 characters)
- [ ] Subtitle (max 30 characters)
- [ ] Description (max 4000 characters)
- [ ] Keywords (max 100 characters)
- [ ] Support URL
- [ ] Marketing URL (optional)
- [ ] Privacy Policy URL (required)

### Screenshots Required

iPhone 6.7" (iPhone 15 Pro Max):
- [ ] 3-10 screenshots (1290 x 2796)

iPhone 6.5" (iPhone 11 Pro Max):
- [ ] 3-10 screenshots (1242 x 2688)

iPhone 5.5" (iPhone 8 Plus):
- [ ] 3-10 screenshots (1242 x 2208)

### App Icon
- [ ] 1024x1024 PNG
- [ ] No transparency
- [ ] No rounded corners

### App Privacy
- [ ] Privacy policy URL added
- [ ] Data collection practices declared
- [ ] Privacy nutrition label completed

### Build Requirements
- [ ] Version number set
- [ ] Build number incremented
- [ ] Archive uploaded to App Store Connect
- [ ] Build processing complete
- [ ] Build selected for version

### Age Rating
- [ ] Completed age rating questionnaire
- [ ] Appropriate rating assigned

### App Review Information
- [ ] Contact information
- [ ] Demo account credentials (if login required)
- [ ] Review notes (if needed)

### Categories
- [ ] Primary category selected
- [ ] Secondary category selected (optional)

## Technical Requirements

### Code
- [ ] No placeholder content
- [ ] No test/debug code in production
- [ ] No console logs in release build
- [ ] All features functional
- [ ] Performance optimized

### Content
- [ ] No offensive content
- [ ] No copyright violations
- [ ] Follows Apple guidelines
- [ ] In-app purchases configured (if applicable)

### Testing
- [ ] Tested on multiple devices
- [ ] Tested on iOS minimum version
- [ ] No crashes
- [ ] All flows working
- [ ] TestFlight beta testing done

## Submission Steps

1. Archive app in Xcode
2. Upload to App Store Connect
3. Wait for processing
4. Complete all metadata
5. Add screenshots
6. Select build
7. Submit for review
8. Wait 24-48 hours for review

## Common Rejection Reasons

- Incomplete information
- Crashes or bugs
- Poor user interface
- Broken links
- Missing privacy policy
- Incomplete app functionality
- Placeholder content
- Copyright issues

## After Approval

- [ ] App goes live or scheduled release
- [ ] Monitor crash reports
- [ ] Check user reviews
- [ ] Respond to feedback
- [ ] Plan next update

## Tips for Faster Approval

Use clear screenshots showing features. Write detailed review notes. Provide working demo account. Test thoroughly before submission. Follow Human Interface Guidelines.
